Real Time Chat-Applcation

Implementation of Chat Application. It is a Web Application which allows transmission of text messages and location in a Full Duplex way.

Sockets IO has been used for a smooth real time transmission of text messages.

Features: Providing Social Platform to Users, To allow each person to share their thoughts & views, Provision of Different rooms for Different Groups, Middleware Implementation for Profanity surveillance.

Role: System Architect & Developer.

Technologies:
  • - JavaScript(EcmaScript 6)
  • - Node JS
  • - Express JS
  • - Socket JS
  • - API
  • - HTML5/CSS3
